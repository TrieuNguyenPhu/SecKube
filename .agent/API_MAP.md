# API Map

Base port inside the container: `8080`. The Kubernetes Service exposes it as port `80` inside the cluster.

| Method | Path | Contract | Operational use |
|---|---|---|---|
| `GET` | `/` | Returns JSON with the deployed version. May return HTTP 500 with an injected-error payload. | Canary test traffic |
| `GET` | `/healthz` | Returns `ok` with HTTP 200. | Liveness and readiness probes |
| `GET` | `/metrics` | Prometheus text exposition generated by the Flask exporter. | ServiceMonitor scrape target |

## Runtime inputs

| Variable | Meaning |
|---|---|
| `VERSION` | Release identifier included in `/` responses |
| `ERROR_RATE` | Probability from 0 to 1 that `/` returns an injected HTTP 500 |

There is no authentication, persistent state, public ingress or write API. Exact metric names and labels come from the exporter and deployment manifests; inspect those sources before changing PromQL.
